相关内容

  • nvm & npm & nodejs

    • nvm：nodejs版本管理工具。nvm可以管理很多node版本和npm版本
    • npm：安装的nodejs的时候，npm会被装上，它是包管理工具。npm管理nodejs中的第三方插件
    • nodejs： 项目开发时的所需要的代码库

    nvm、nodejs、npm的关系：

    nvm是爸爸，管理nodejs和npm这一对双胞胎兄弟。npm是哥哥，npm哥哥可以管理node弟弟的东西。

  • n & nvm

    • n：n是一个需要全局安装的 npm package。因此在安装n前需要已有node环境

    • nvm：独立的软件包，单独安装

    nvm、n区别：

    ​ 1.nvm 不支持 Windows；

    ​ 2.n 对全局模块毫无作为，因此有可能在切换了 node 版本后发生全局模块执行出错的问题；nvm 的全局模块存在于各自版本的沙箱中，切换版本后需要重新安装，不同版本间也不存在任何冲突；

    ​ 3.node 路径。n 是万年不变的 /usr/local/bin；nvm 需要手动指定路径。
